About Task Manager

Hello welcome to Task Manager! Below you can find information about all the functionalities of this system.

The Home Page contains all the redirection links to every other function in the system - Calendars, Reports, About page, Contacts page and Settings. You can also log out using the button at the bottom. There is also a timer at the top that shows current time.

Calendar pages allow you to plan specific events. You can search up specific event through a text field in the left corner, or go to today's date by clicking the button "today". You can switch the months using arrows. By clicking on a specific day you select it as an active day. On the right side, you can see a "+" symbol, after clicking it you are able to set up an event. Events are marked with an underline on a specific day. Through clicking on a given event on the right side, you can delete it.

The To-Do Planner page allows you to freely add every task that has to be done, regarding all calendars. From that, you can navigate to the calendars and assign specific tasks to specific field using categories - Work or Personal. You can add a task by putting information at the top. To delete or edit or mark task as assigned, click on the corresponding buttons next to tasks. To mark task as done, select the radio button next to a task.

Lastly, the Reports page allows you to write daily reports about the task work that has been done and send it to a greater database.
